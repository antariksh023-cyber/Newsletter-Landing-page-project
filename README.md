# Newsletter Landing Page

![Preview](pfp.png)

A clean, elegant, and minimalist **newsletter subscription landing page** built with pure HTML and CSS. Perfect for creators, developers, bloggers, or anyone who wants a professional-looking signup page in seconds.

## ✨ Features

- Modern dark theme with smooth hover effects
- Large circular profile picture (uses your `pfp.png`)
- Responsive-ish centered layout (works great on desktop)
- Clean typography using system fonts
- Fully functional-looking form (ready for you to connect to Mailchimp, ConvertKit, Buttondown, etc.)
- Custom favicon (same `pfp.png`)
- Zero dependencies — just one HTML file!

## 🖼️ Live Demo

You can host this instantly for free on **GitHub Pages**, **Netlify**, or **Vercel**.

## 🚀 How to Use

1. Download the repository (or clone it)
2. Make sure you have your profile picture saved as **`pfp.png`** in the root folder
3. Open `Newsletter landing page.html` in your browser
4. (Optional) Connect the form to your newsletter service by adding an `action` attribute to the `<form>` tag
